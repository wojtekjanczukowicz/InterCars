# InterCars - Playwright Test Automation

## Opis
Automatyczny test scenariusza zakupowego dla strony InterCars.

## Zakres testu
- wejście na stronę
- akceptacja cookies
- wyszukanie produktu
- dodanie do koszyka
- weryfikacja ceny

## Technologie
- Playwright
- TypeScript

## Uruchomienie
npm install
npx playwright test

## Uwagi
- strona zawiera captcha – test zawiera workaround
- dynamiczne selektory mogą powodować niestabilność
